# shell命令

`man`  查看命令手册

`file`  查看文件类型

`cat more less` 查看文件

`tail`  查看末尾  tail -n 2 xxx

`ps -ef` `ps aux`  监控进程

##### tar命令

```bash
tar -cvf xx.tar f1 f2 ...
tar -cvzf xx.tar.gz f1 f2 ...
tar -xvf xx.tar
tar -xvzf xx.tar.gz
tar -tvf xx #查看内容
```

`jobs -l `  显示后台进程

`coproc <command>`  启动协程，后台执行命令

`alias ll='ls -al'`  给命令起别名
